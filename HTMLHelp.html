If your webpage is not displaying as you expected in the web browser here is what you should check

  1. Check you have spelt the tag correctly <img> is NOT the same as <igm>
  2. Does the HTML tag use attributes, i.e. the image tag
    <img> src="" is NOT the same as
    <img> scr=""
  3. Have you got the tags in the correct order see structure below
    		<HTML>
			<head>
				header text - css & javascript
			</head>
			<body>
				tag then add your code/text for you webpage
			</body>
		</HTML>
